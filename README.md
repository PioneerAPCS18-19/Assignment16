# Assignment 16

Given a string, if the first or last chars are 'x', return the string without those 'x' chars, and otherwise return the string unchanged.

### Sample Outputs

```
withoutX("xHix") → "Hi"
withoutX("xHi") → "Hi"
withoutX("Hxix") → "Hxi"
withoutX("Hxi") → "Hxi"
withoutX("") → ""
withoutX("x") → ""
```
